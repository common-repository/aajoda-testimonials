Integrate Aajoda Testimonials on your WordPress



== Description ==

This plugin uses a shortcode [aajoda id="xxxxxxxxx"] which displays the testimonials and reviews from your Aajoda-Account.

You need to create a free account with a connected company to be able to use this plugin.




== Installation ==


1. Upload the `aajoda-testimonials` folder and all its contents to the `/wp-content/plugins/` directory

2. Activate the plugin through the 'Plugins' menu in WordPress

3. Get the custom shortcode [aajoda] generated from your [aajoda.com](https://www.aajoda.com/) dashboard
and place it where you want to display it.
